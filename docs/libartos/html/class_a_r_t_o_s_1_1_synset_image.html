Detailed Description

Stores information (image data and bounding box annotations) about an image in a synset of of an image repository.

Author
Bjoern Barz bjoer.nosp@m.n.ba.nosp@m.rz@un.nosp@m.i-je.nosp@m.na.de

bool SynsetImage::loadBoundingBoxes ( )

Loads bounding box annotations for this image into the bboxes vector if available.

The bounding boxes given in the file will be rescaled according to the size of this image.

Successive calls to this function will do nothing.

Returns
Returns true if bounding box annotations are available for this image, otherwise false.
