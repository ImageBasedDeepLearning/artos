Detailed Description

The Rectangle class defines a rectangle in the plane using integer precision. If the coordinates of the top left corner of the rectangle are (x, y), the coordinates of the bottom right corner are (x + width - 1, y + height - 1), where width and height are the dimensions of the rectangle. The corners are thus understood as the extremal points still inside the rectangle.
